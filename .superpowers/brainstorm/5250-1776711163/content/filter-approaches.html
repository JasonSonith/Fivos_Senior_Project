Filter UI Approaches

How should clickable metric cards look and behave on the Dashboard?

A

Active Card Highlight

Cards get a colored left border + subtle background tint when active. An "All" pill button appears above the table when a filter is active. Clicking the same card again deselects it (returns to All). Minimal visual change — stays true to current design.

B

Card as Toggle Button

Cards gain a hover cursor and ring effect on hover. Active card gets a bold teal ring + "FILTERING" micro-label. Non-active cards stay normal (no dimming). A floating "Clear filter" link appears next to the table header. More explicit affordance that cards are clickable.

C

Separate Filter Bar

Cards stay as pure display (no click behavior). Instead, add a pill/tab filter bar between the cards and the table: All | Matched | Partial | Mismatch. Cleanest separation of concerns — stats above, filter below, table at bottom. Most conventional pattern.
